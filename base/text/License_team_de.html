Ipponboard Team Edition v1.0

Dies ist keine freie Version. Kopieren und Weitergeben ist untersagt.

Wenn Sie Ipponboard gut finden, sollten Sie über eine Spende nachdenken, um die Weiterentwicklung und Pflege auch längerfristig sicherzustellen. Spenden können Sie einfach über den Link auf der Homepage oder Sie kontaktieren uns einfach.

Falls Sie Verbesserungsvorschläge bzgl. der Anzeige (Darstellung, Bedienung) oder der Steuerung haben, oder aber finden, dass es noch irgendwo hakt, dann teilen Sie uns dies bitte mit - wir freuen uns, wenn wir unsere Software noch besser machen können. Hierzu gibt es einen Online-Fragebogen, erreichbar über Menü→Über→Feedback (dauert wirklich nicht lange).

Die Version ist bis Ende 2012 gültig. Danach wird sie durch eine neuere Version ersetzt.

Die neueste Version gibt es im Netz unter: http://www.ipponboard.info
